# Triple Espresso

Este projeto é sobre uma página web de uma cafeteria chamada Web Café, onde é possível conhecer o ambiente, algumas receitas e também reservar mesas em caso de reuniões ou trabalhos presenciais remotos.

Utilizamos técnicas como o <iframe> para demonstrar receitas do cardápio, o <form></form> para a reserva de mesas e tudo dentro das técnicas do BEM.

Como melhoria, ou sugestão de melhoria, a página de reservas poderia compreender todo a página, dessa forma reduzindo algumas medidas de altura do formulário ou padding para que tudo fique dentro dos 800 pixels, assim quando o usuário usar o link do nav para reservar a mesa, toda a tela dele mostrará todos os dados necessário, sem ele precisar rolar a página para concordar com os termos.
